Trabajo Práctico Integrador

Este fue el trabajo práctico realizado para aprobar la materia Programación 2. El programa consiste en un sistema de gestión de turnos para una clínica, la cual permite agregar, modificar y eliminar pacientes, profesionales y especialidades. Además cuenta con validaciones para evitar fechas inexistentes y errores en el ingreso de datos. Se encuentra desarrollado completamente en C++ con Code::Blocks.
